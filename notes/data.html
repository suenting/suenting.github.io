Linear Discriminant Analysis

Linear Discriminant Analysis, is used to solve classification problems by mapping a target input vector (TI) to a target output vector (TO).
Where TI is an M dimensional input vector. and TO is and N dimensional output vector
This is done by solving the linear equation. Which divides the M dimensional space via a hyperplane to one of N segments

TO = TI*W0 + C

Where W0 is a weight matrix, and C is a constant vector.
Additionally W0 and C are often derived from a learning data set by a collection of TI and resulting TO, (existing data) such that the TO can be predicted for future TI with this method.

K-Means

K-Means, is a method to solve classification problems by mapping entities in a data set into one of k-groups. The algorithmn is as follows:
  • Given K centers (with an arbitary initial position) in the dataset
  • Map all data points to the nearest center.
  • Update the position of all centers to the averaged position of all data points which are mapped to it.
  • Repeat untill convergance. (or an arbitary cut of itteration)
Note: The above is not gaurenteed to converge, and the resulting classification is affected by the initial positions of the K centers.
